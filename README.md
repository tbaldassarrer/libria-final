# Libria

Aplicación web de biblioteca desarrollada con Spring Boot.

## Autores

- [tbaldassarrer](https://github.com/tbaldassarrer)
- [YexeniaRosales](https://github.com/YexeniaRosales)

## Descripción

Libria es una plataforma para gestionar libros, usuarios, reseñas y más.

## Tecnologías

- Java 17
- Spring Boot 3.4.0
- MySQL
- Thymeleaf
- Maven

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tbaldassarrer/libria-final.git
   ```

2. Configura la base de datos en `application.properties`.

3. Ejecuta con Maven:
   ```bash
   mvn spring-boot:run
   ```
